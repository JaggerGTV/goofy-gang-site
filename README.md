
# Goofy Gang Website

## Setup

1. Install Node.js
2. Open terminal in project folder
3. Run:

npm install

4. Start development server:

npm run dev

5. Open the local URL shown in terminal

---

## IMPORTANT

Replace:

CHANNEL_ID

inside src/App.jsx with your real YouTube channel ID.

---

## Add Hero Image

Put your image file here:

public/super-joshy.png

---

## Upload to GitHub

1. Create a new GitHub repository
2. Upload all files
3. Push changes

---

## Deploy to Render

1. Go to https://render.com
2. New Web Service
3. Connect GitHub repo
4. Build Command:

npm run build

5. Publish Directory:

dist

6. Deploy
